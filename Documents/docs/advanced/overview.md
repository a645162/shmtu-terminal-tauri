# 高级版本总览

这部分不是给第一次使用的人看的，而是给下面这几类人：

- 想真正理解程序结构的人
- 需要排查复杂问题的人
- 需要维护、修改或长期使用这套程序的人

## 这部分会讲什么

- 前端和 Tauri 后端怎么分工
- 全局状态怎么组织
- 同步链路怎么跑
- 验证码模式怎么接进来
- 配置、数据库、快照落在哪里
- 怎么构建和发布文档

## 推荐阅读顺序

1. [应用结构](/advanced/app-structure)
2. [数据流与状态](/advanced/data-flow)
3. [同步与验证码](/advanced/sync-and-captcha)
4. [配置与存储](/advanced/config-and-storage)
5. [构建与发布](/advanced/build-and-release)

## 和详细说明的区别

- `普通用户版`：只讲怎么用
- `详细说明`：用户视角的完整说明
- `高级版本`：讲程序机制和结构
