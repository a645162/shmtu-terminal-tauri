# 界面总览

如果你刚打开软件，不知道该从哪里点起，这页就是地图。

## 先认三个主要入口

顶部最重要的就是这三块：

- `首页`
- `账单`
- `功能大全`

右侧还包含：

- 当前身份徽标
- 主题切换
- 身份菜单
- 更多菜单

## 首页是“今天先看一眼”的地方

首页适合快速确认最近有没有新数据，重点展示：

- 今日消费
- 本月消费
- 本月充值
- 趋势图
- 分类占比
- 忘刷卡提醒
- 最近账单

如果你只是想知道“最近有没有同步成功、最近花了多少”，通常先看首页就够了。

## 账单页是“真正动手操作”的地方

大部分用户操作都会落在账单页，包括：

- 按状态筛选
- 按时间筛选
- 关键词搜索
- 分页浏览
- 查看账单详情
- 修改备注
- 删除合并账单
- 去重、重建
- 发起身份级或账号级同步

## 功能大全是“找不到入口时去的地方”

它把一些次级功能集中放在一起：

- 统计分析
- 数据导出
- 数据导入
- 验证码测试
- 身份管理
- 应用设置
- 数据快照
- 关于
- 检查更新

如果你忘了“统计”“导入导出”“验证码测试”这些功能在哪里，就去这里找。

## 底部状态栏用来确认你现在在哪个身份

底部会显示：

- 当前身份
- 程序版本
- 技术栈标识

切换身份后，先看这里有没有变；没变通常就说明切换还没生效。

## 你会经常遇到的弹窗

程序很多功能不是跳新页面，而是直接弹窗打开，常见有：

- 身份选择弹窗
- 身份与账号管理弹窗
- 设置弹窗
- 数据传输弹窗
- 统计弹窗
- 手动验证码弹窗
- OCR 模型下载弹窗
