# 界面总览

主界面由 `顶部导航 + 主内容区 + 底部状态栏` 组成。

## 顶部区域

顶部主要包含三部分：

- `首页`
- `账单`
- `功能大全`

右侧还包含：

- 当前身份徽标
- 主题切换
- 身份菜单
- 更多菜单

## 首页

首页适合日常快速查看，重点展示：

- 今日消费
- 本月消费
- 本月充值
- 趋势图
- 分类占比
- 忘刷卡提醒
- 最近账单

如果你只想判断“最近有没有同步到新数据”，首页通常就够用了。

## 账单页

账单页是最常用的操作页，主要能力包括：

- 按状态筛选
- 按时间筛选
- 关键词搜索
- 分页浏览
- 查看账单详情
- 修改备注
- 删除合并账单
- 去重、重建
- 发起身份级或账号级同步

## 功能大全

功能大全聚合了次级功能入口：

- 统计分析
- 数据导出
- 数据导入
- 验证码测试
- 身份管理
- 应用设置
- 数据快照
- 关于
- 检查更新

它更像一个“控制台”，适合在不记得入口位置时快速进入功能。

## 底部状态栏

底部状态栏会显示：

- 当前身份
- 程序版本
- 技术栈标识

如果你在切换身份后发现状态栏没有变化，通常说明身份切换没有真正完成。

## 常见弹窗

程序的大部分高级功能都以弹窗形式打开，常见有：

- 身份选择弹窗
- 身份与账号管理弹窗
- 设置弹窗
- 数据传输弹窗
- 统计弹窗
- 手动验证码弹窗
- OCR 模型下载弹窗
