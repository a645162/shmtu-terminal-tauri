# 常见问题

## 程序启动后没有任何账单

先依次检查：

1. 是否已选择身份
2. 身份下是否真的挂了账号
3. 账号是否启用同步
4. 是否已经执行过同步

## 同步时一直要求验证码

这通常不是异常，而是你当前选择了人工验证码模式，或者账号会话已经失效。

建议：

1. 先用手动验证码完成一次同步
2. 再检查 OCR 模式是否可用

## 本地 ONNX 模式无法工作

优先检查：

- 模型是否完整下载
- 模型路径是否正确
- 本地环境是否能正常加载 ONNX Runtime

如果已经出现模型损坏提示，最稳妥的方式是删除本地模型并重新下载。

## 统计结果看起来不对

优先排查：

- 历史同步是否完整
- 是否有重复账单
- 分类规则是否需要更新

不要直接把统计异常等同于原始数据损坏。

## 切换身份后内容没有刷新

通常是以下原因之一：

- 身份没有真正切换成功
- 账单列表仍停留在旧分页条件
- 账号列表还未重新加载

建议重新选择身份后，再手动刷新账单页。

## 导入后出现重复记录

这是典型的导入后整理问题，不一定是导入失败。

建议流程：

1. 先做账号级或身份级去重
2. 必要时执行重建
3. 重新核对统计结果

## 快照恢复后程序行为和之前不一致

先确认恢复的快照时间点是否正确。快照恢复的不只是账单，还可能包含配置和状态变化。

## 需要截图时应该截哪些页面

建议优先保留：

- 主界面首页
- 账单页
- 统计弹窗
- 设置页
- 手动验证码弹窗
- OCR 模型下载弹窗
