# 怎么同步账单

大多数时候，你只需要记住这三种同步：

- `增量同步`
- `全量同步`
- `单账号同步`

## 平时用哪种

平时更新最近账单，直接用：

- `增量同步`

这是最常用的方式。

## 什么时候用全量同步

这些情况再用：

- 第一次使用
- 历史数据明显不全
- 你想重新补一段较久以前的记录

## 什么时候用单账号同步

如果一个身份下有多个账号，而只有一个账号有问题，就只同步那个账号。

这样更快，也更省事。

## 验证码怎么选

第一次推荐：

- `手动验证码`

已经稳定使用后，再考虑：

- 远程 OCR
- 本地 OCR

## 如果同步时停住了

先看它是不是在等验证码。

如果界面提示要验证码，通常不是坏了，只是在等你继续操作。

## 最简单的建议

- 平时更新：增量同步
- 补历史：全量同步
- 某个账号异常：单账号同步
- 第一次使用：先手动验证码
