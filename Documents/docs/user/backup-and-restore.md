# 怎么备份和恢复

如果你要导入数据、恢复旧数据、或者准备做大调整，先看这页。

## 最重要的一条

做大操作前，先创建快照。

## 什么情况下要先备份

建议在这些操作前先做一次快照：

- 导入数据
- 大范围清理数据
- 尝试恢复旧内容
- 重新整理大量账单

## 快照有什么用

你可以把它理解成：

- 给当前状态拍一张“存档”

如果后面操作结果不满意，还可以退回来。

## 导出适合什么时候用

- 想自己留一份数据
- 想拿去表格里看
- 想导到别的记账工具

## 恢复后要检查什么

恢复完成后，建议马上看三件事：

- 当前身份对不对
- 账单数量对不对
- 最近统计看起来正不正常

## 最简单的习惯

- 动数据前先快照
- 恢复完先检查
