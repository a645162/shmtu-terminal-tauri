# 怎么看统计

如果你只是想快速看最近花了多少，先看首页；想深入分析消费结构，再去统计页。

## 首页适合看什么

首页是"打开软件第一眼"的地方，重点展示快速概览信息：

- **今日消费**：当天累计消费金额
- **本月消费**：当前月份累计消费金额
- **本月充值**：当前月份累计充值金额
- **消费趋势缩略图**：近期每日消费走势
- **分类占比缩略图**：消费按分类的大致比例
- **忘刷卡提醒**：识别出的可疑消费记录
- **最近账单**：最近几笔消费记录

如果你只是想知道"最近有没有同步成功、最近花了多少"，通常看首页就够了。

## 统计页适合看什么

统计页（从功能大全进入）提供更完整的分析视图，适合做消费结构和趋势判断：

- **消费总览**：选定时间段内的消费总额、笔数、日均
- **每日趋势图**：按天展示消费变化，识别异常日
- **分类分布图**：饼图或柱状图展示各分类占比
- **用餐时段分布**：早/中/晚/夜各时段的消费分布
- **消费金额分桶**：按金额区间统计消费频次
- **商户消费排行**：哪些商户消费最多、频次最高
- **忘刷卡统计**：特定模式的可疑消费记录汇总
- **分类明细账单**：点击某个分类后，查看该分类下的具体账单

## 时间范围怎么选

统计页支持多种时间范围，选择合适的范围能更好地发现规律：

| 时间范围 | 适合看什么 |
|----------|-----------|
| 今日 | 当日消费与异常 |
| 本周 / 近 7 天 | 近期消费变化 |
| 本月 | 阶段消费习惯 |
| 季度 / 半年 / 一年 | 长期趋势 |
| 全部 | 历史总览 |

日常使用建议从"本月"开始看，发现异常后再缩小或扩大范围。

## 如何解读分类结果

分类统计依赖本地规则文件（`classification_rules.toml`），它把商户名和交易描述归到不同类别。

需要理解的是：

- 分类结果更适合**看趋势和占比**，而不是当作财务绝对真值
- 如果某些交易分类不符合预期，通常不是账单丢失，而是规则命中方式需要调整
- 规则文件可以从远端更新，更新后需要执行"重建"才能让统计反映新的分类

## 如果图表看起来不对

先不要急着怀疑程序坏了，按以下顺序排查：

1. **同步是否完整**：最近有没有真正同步完整？增量同步可能只更新了部分数据
2. **有没有重复账单**：重复记录会让金额偏高，去重后再看统计
3. **时间范围是否正确**：确认你选的统计范围和你想看的时段一致
4. **分类规则是否加载**：如果分类全显示"未分类"，说明规则文件可能缺失

## 最简单的理解方式

- 首页：快速看，回答"最近花了多少"
- 统计页：详细看，回答"钱花在哪了"
