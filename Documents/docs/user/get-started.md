# 第一次使用

如果你是第一次打开软件，按下面做就够了：

1. 打开软件
2. 新建一个身份
3. 在身份下面添加账号
4. 先选 `手动验证码`
5. 做一次同步

## 你要先准备什么

- 学号或账号
- 对应密码

只要这两样有，其他都可以后面再调。

## 第一步：先建身份

如果你只有自己一个人要用，身份名直接写成自己的名字就行。

## 第二步：添加账号

把账号、密码填进去，保存即可。

如果你有多个账号，也可以一个身份下继续加，但第一次不必一步到位。

## 第三步：先选手动验证码

第一次最重要的是先跑通。

所以建议先选：

- `手动验证码`

等你确认同步没问题，再考虑切到 OCR。

## 第四步：做第一次同步

如果你只想先试一下：

- 点 `增量同步`

如果你想把历史数据多补一点：

- 点 `全量同步`
- 范围选 `最近半年`

## 第五步：看结果

同步完成后，你可以这样确认：

- 首页能看到统计卡片
- 账单页能看到记录
- 当前身份显示正确

## 先记住两条

- 大操作前先做快照
- 不懂的设置先别改
